🗄️ ESP32 Unified Cloud Drive (v2.0 Pro)ESP32 Unified Cloud Drive is a standalone, secure, local cloud storage solution built on the ESP32 microcontroller. It features a stunning glassmorphism web interface for local network access and a fully integrated Telegram Bot for remote file management from anywhere in the world.✨ FeaturesSecure Web Dashboard: Password-protected portal with pseudo-random session tokens to prevent unauthorized access.Modern UI/UX: Responsive, dark-themed "glassmorphism" web interface with animated backgrounds and intuitive file management.Dual-Access System: * Local: High-speed file uploads, downloads, and deletions via the Web UI.Remote: Manage files globally via the Telegram Bot API.Storage Analytics: Real-time SD card statistics (Total, Used, and Free space) with visual progress bars.OLED Integration: Live system status, boot animations, and active process monitoring on a 128x64 I2C OLED display.Intrusion Alerts: Logs unauthorized Telegram access attempts and sends instant security alerts to the admin.🛠️ Hardware RequirementsESP32 Development Board (NodeMCU-32S, WROOM-32, etc.)MicroSD Card Adapter Module (SPI Interface)MicroSD Card (Formatted to FAT32)0.96" OLED Display (SSD1306, I2C Interface)Jumper wires and breadboardWiring GuideComponentPin NameESP32 PinSD Card ModuleCS (Chip Select)GPIO 5MOSIGPIO 23MISOGPIO 19SCK (Clock)GPIO 18VCC / GND3.3V or 5V / GNDOLED DisplaySDAGPIO 21SCLGPIO 22VCC / GND3.3V / GND📦 Software DependenciesEnsure you have the following libraries installed in your Arduino IDE via the Library Manager:Adafruit GFX Library (by Adafruit)Adafruit SSD1306 (by Adafruit)UniversalTelegramBot (by Brian Lough)ArduinoJson (by Benoit Blanchon - required for Telegram Bot)Note: WiFi.h, WebServer.h, SPI.h, SD.h, and Wire.h are built-in with the ESP32 board package.🚀 Setup & Installation1. Prepare the Telegram BotOpen Telegram and search for @BotFather.Send /newbot and follow the prompts to create your bot.Save the HTTP API Token provided by BotFather.Search for @IDBot on Telegram and send /getid to find your personal Chat ID.2. Configure the CodeOpen the .ino file in the Arduino IDE and navigate to SECTION 1 ── CONFIGURATION. Update the following variables with your details:C++// Network
const char* WIFI_SSID     = "YOUR_WIFI_SSID";
const char* WIFI_PASSWORD = "YOUR_WIFI_PASSWORD";

// Telegram
const char* BOT_TOKEN = "YOUR_BOT_TOKEN_HERE";
const char* CHAT_ID   = "YOUR_CHAT_ID_HERE";

// Web Dashboard Credentials
const char* WEB_USERNAME = "admin";
const char* WEB_PASSWORD = "your_secure_password"; 
3. Flash to ESP32Connect your ESP32 to your computer.Select the correct COM port and Board (e.g., "DOIT ESP32 DEVKIT V1").Compile and upload the code.The OLED display will show the boot animation, mount the SD card, and display your local IP address once connected.📱 Usage GuideWeb Dashboard (Local Network)Ensure your device is on the same WiFi network as the ESP32.Open a web browser and navigate to the IP address shown on the OLED display (e.g., http://192.168.1.100).Log in using your configured WEB_USERNAME and WEB_PASSWORD.Use the dashboard to upload files, download items, monitor storage space, and delete files.Telegram Bot (Remote Access)Send messages to your bot on Telegram using the following commands:/start - Displays the welcome message and available commands./list - Lists all files currently stored on the SD card with their sizes./stats - Shows a visual breakdown of SD card storage./send <filename> - Generates a direct download link for a specific file./delete <filename> - Permanently deletes a file from the SD card./ip - Retrieves the current local IP address of the web server./reboot - Restarts the ESP32 remotely.🔒 Security NotesSession Management: The web server uses pseudo-random session tokens linked to the ESP32's MAC address and uptime. Sessions expire automatically after 30 minutes.Telegram Chat ID Lock: The bot will reject commands from any Telegram user whose Chat ID does not match the one hardcoded in the configuration. It will also notify the admin of the attempted breach.👨‍💻 AuthorDesigned and developed by Md Nazmun Nur.
